market.market module¶

market.market.PRODUCT_PRICES¶

A mapping of product codes to their prices.

market.market.DISCOUNT_AMOUNTS¶

A mapping of discount codes and the amount that they reduce the register total by, as a function of PRODUCT_PRICES.

market.market.CURRENT_DISCOUNTS¶

A list of currently-applicable discounts of the form list(function).

Example:

CURRENT_DISCOUNTS = [bogo, appl]

This would make it so the unlimited buy-one-get-one-free special on coffee and the purchase a bag of oatmeal and get 50% off a bag of apples discount are applied during register().

market.market.apply_discount(trigger, triggers_needed, affected, discount, when, basket)[source]¶

Adds the given discount to the basket, if all preconditions are met, returning the new basket with discounts present.

This is the function that all discounts are expected to call, hence the deprecation of the *_verbose functions.

Example:

>>> market.PRODUCT_PRICES = {'FOO': 4, 'BAR': 2}
>>> market.DISCOUNT_AMOUNTS = {'BAZ': -3.50}
>>> def quux(basket):
...     return market.apply_discount('BAR', 1, 'BAR', 'BAZ',
...                                  lambda *args: True, basket)
...
>>> setattr(market, 'quux', quux)
>>> market.CURRENT_DISCOUNTS = [quux]
>>> market.register(['FOO', 'FOO', 'BAR', 'BAR'])
[('FOO', 4), ('FOO', 4), ('BAR', 2), ('BAZ', -3.5), ('BAR', 2),
 ('BAZ', -3.5)]
>>> market.register(['FOO', 'BAR'])
[('FOO', 4), ('BAR', 2), ('BAZ', -3.5)]
>>> market.register(['BAR', 'FOO', 'BAR'])
[('BAR', 2), ('BAZ', -3.5), ('FOO', 4), ('BAR', 2), ('BAZ', -3.5)]
>>>
Parameters:
  • trigger (str) – The product code that potentially triggers a discount.
  • triggers_needed (int) – The minimum number of triggers that must be seen in basket to potentially apply a discount.
  • affected (str) – The product code that is affected by the trigger, which determines the insertion position of discount.
  • discount (str) – The product code representing the discount.
  • when (function) –

    The function to call to ultimately determine whether or not to apply the discount. This function receives two arguments, in the following order:

    • seen_count: an int representing the number of times that affected has been seen
    • affected_count: an int representing the number of times that the discount has already been applied in the currently executing context of apply_discount()
  • basket (list(str)) – The basket to apply discount to.
Returns:

The basket with the applicable discount applied.
Return type:

list(str)

market.market.register(basket)[source]¶

Returns a register for the given basket, which is a representation of codes and their associated values.

Example:

>>> market.register(['BAR', 'FOO', 'BAR'])
[('BAR', 2), ('BAZ', -3.5), ('FOO', 4), ('BAR', 2), ('BAZ', -3.5)]
Parameters:basket (list(str)) – The basket to create a register from.
Returns:The basket with all amounts associated to their codes.
Return type:list(tuple(str, float))
